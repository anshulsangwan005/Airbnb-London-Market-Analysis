# Airbnb-London

**Overview:**

This project analyses the Airbnb listings in London to uncover insights about pricing, availability, room types, and neighborhood-wise trends. The goal is to understand the London Airbnb market using exploratory data analysis (EDA) and visualizations.

The analysis helps answer questions like:

Which areas are the most expensive or affordable?
How does price vary by room type?
What patterns exist in availability and reviews?
Which neighborhoods have the highest Airbnb activity?
